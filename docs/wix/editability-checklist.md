# Editability Checklist

## 目的
- Wix Studio へ移した後に、最小ページが再編集しやすい構造になっているかを確認する
- テキスト差し替え、画像差し替え、セクション順変更、CTA 変更を先に観点化する

## 確認観点

### テキスト差し替え
- 各セクションの見出しと本文が分かれている
- 見出し階層が `h1 -> h2 -> h3` の順で整理されている
- 本文差し替え箇所がまとまっており、他要素に巻き込まれにくい

### 画像差し替え
- 画像領域が独立したブロックになっている
- 画像差し替え時に隣接テキストの構造を壊さずに済む
- 画像サイズ変更の影響範囲がセクション内で閉じている

### セクション順変更
- セクション境界が明確で、役割ごとに分かれている
- Hero、Overview、Explanation、CTA の順を入れ替えても意味が崩れにくい
- セクション内の要素が他セクションに依存していない

### CTA 変更
- CTA 文言の差し替え箇所が明確
- CTA リンク先を独立して変更できる
- 主要 CTA と補助 CTA が視覚的に区別されている

## 最小ページで見るポイント
- `prototype/minimum-page/index.html` に section ごとの境界と役割ラベルがある
- 見出し、本文、CTA、画像プレースホルダが分離されている
- Wix Studio 上で一部差し替え後も、再編集と再配置が続けられる構造になっている
