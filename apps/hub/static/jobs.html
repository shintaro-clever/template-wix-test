Hub Jobs Console (Fallback)

npm install 不要。Offline smoke → Docs Update → Repo Patch をここで生成し、run.json / audit.jsonl を貼り付けて Gate / Triage を確認できます。

絶対ルール：本番ジョブ前に offline smoke（local_stub）を必ず実行する。

Job Builder

まず Offline smoke ジョブを生成して保存 → `node scripts/run-job.js --job ... --role operator` を実行し、成功したら Docs Update → Repo Patch の順に進めます。

次に実行するコマンド node scripts/run-job.js --job <job-json-file> --role operator

Run Result Intake

`.ai-runs//run.json`（必須）と `audit.jsonl`（任意）を貼り付けて Parse を押すと、Gate / Triage をブラウザで確認できます。推奨入力は run.json（必須）と audit.jsonl（任意）の組み合わせです。
